This REST API Service is to provide the customer the following information:

    1. A Working RESTful Web Service listening on http://localhost:8080
       a. Curl returns

    2. A Service that Accepts a postive number as input and returns Fibonacci numbers at http://localhost:8080/fib/$number via POST.
       a. example would be http://localhost:8080/fib/5 to return 5 integers starting with 0 and 1 as

           curl -x POST http://localhost:8080/fib/5
          {
             "result": [
               0,
               1,
               1,
               2,
               3
             ]
           }

The following are expected and should return an error

    1. When the customer enters a negative or alpha value as a Fibonacci number.
       a. example would be http://localhost:8080/fib/-5 resulting in an 404 status code.
